# Overview

## Purpose

A specialized CRM system built for fullstack web developers and digital agencies serving German and EU clients. Unlike traditional CRM solutions focused on sales pipelines and lead management, this application prioritizes:

- **Automated Document Generation**: Eliminate manual quote and invoice creation in Word/Excel
- **German Legal Compliance**: Full GoBD (Grundsätze zur ordnungsmäßigen Führung und Aufbewahrung von Büchern) compliance for invoicing
- **Developer Workflow**: Streamlined client/project management tailored for web development services
- **Professional Communication**: Automated email delivery with PDF attachments

### Key Differentiators

- Tailored for web development service businesses, not general sales
- German tax law compliance built-in (Pflichtangaben, sequential numbering)
- Simple, efficient workflow without unnecessary CRM complexity
- Direct PDF generation from database (no manual document editing)
